# Artsite

An art gallery and auction platform built with Ruby on Rails.

![repo_cover](https://user-images.githubusercontent.com/92033807/215205758-daa22277-9b74-4d06-9080-69f069f7adae.jpg)

## Features

- **Art Gallery** - Browse and display paintings, designs, and videos
- **Auctions** - Create and participate in art auctions
- **Bidding System** - Place bids on artworks
- **Watchlist** - Track auctions you're interested in
- **Categories** - Organize artworks by category
- **User Dashboard** - Manage your artworks and auction activity
- **Authentication** - Secure user registration and login

## Tech Stack

- Ruby 3.4.5
- Rails 7.2
- SQLite
- Hotwire (Turbo + Stimulus)
- Devise (authentication)
- Pagy (pagination)
- Ransack (search)
- Solid Queue (background jobs)
- Solid Cache (caching)

### Prerequisites

- Ruby 3.4.5
- SQLite
